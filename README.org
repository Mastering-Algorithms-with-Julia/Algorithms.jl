* Mastering Algorithms with Julia
** 如何贡献代码
首先在Project板块中选取板块，写上自己负责的任务
接下来负责那个模块，编写代码即可

请选取一部分相关内容，比如在 *排序和搜索* 中，有
- 插入排序
- 快速排序
- 归并排序
- 计数排序
- 基数排序
- 二分查找

接下来仿照 Project 中的内容来写任务即可
